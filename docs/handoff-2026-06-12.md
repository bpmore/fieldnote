# Handoff — June 12, 2026

State of the project after the big build day, written so work can resume
from a fresh `git pull` anywhere.

## Get running on a new machine

```
git clone https://github.com/bpmore/fieldnote.git && cd fieldnote
composer install
php bin/audit-themes.php     # expect 70/70
php bin/smoke-test.php       # expect 116 checks, all green (needs ext-zip)
```

Serve `public/` (Herd: drop in ~/Herd and `herd secure`; or
`php -S 127.0.0.1:8000 -t public public/index.php`).

**Your content does NOT travel with git.** `data/` (config, posts,
password, 2FA, passkeys, federation keys) and `public/uploads/` are
gitignored and live only on the original machine. To continue with the
same blog at home, either:

- **Export/import** (built today): dashboard → Export on the old machine,
  run first-time setup on the new one, dashboard → Import the zip
  (posts, tags, drafts, images, post passwords all round-trip), or
- **rsync** `data/` and `public/uploads/` for a byte-identical move
  (brings the admin password, TOTP, and stats too).

Caveats that don't travel regardless: passkeys are bound to the domain
(re-register), and the original dev machine's site lives at
https://fieldnote.test via Herd.

## What shipped today (all on main, CI green)

Everything in `docs/roadmap.md` except 0.6, plus extras. Highlights:

- **Palette customizer** (`/admin/palette`) — recolor any theme; WCAG
  math server-side; failing palettes can't be saved, nearest passing
  shade offered. The headline feature.
- **Theme gallery** (`/admin/themes`) — full-width live light/dark
  previews, one-click apply.
- **Tags, zero-JS search, cron-less scheduled publishing, JSON Feed,
  sitemap, robots.txt** — all with conditional-GET 304s.
- **Content accessibility lint** on save + public `/accessibility`
  statement generated from the enforced constants.
- **Privacy stats** — per-post views, no IPs/UAs ever on disk.
- **Draft share links** (HMAC, 14-day), **post revisions** (last 10,
  restorable).
- **Markdown export/import** with dry-run and no-duplicates guarantee.
- **Passkeys (WebAuthn)** — registered + manually verified working;
  password/TOTP fallback intact. **Session epoch**: password change
  logs out every other device.
- **Hardening**: strict public CSP, trusted-proxy login throttle,
  canonical-host + http→https 301s, image-leak cleanup +
  `bin/prune-images.php`, cached pagination count, feed 304s.
- **ActivityPub AP-1** — followable-but-silent: webfinger, actor,
  signed Follow/Undo inbox with Accept delivery, follower store.
  Off by default (Settings → Fediverse federation).
- **CI**: `bin/smoke-test.php` (116 end-to-end checks) + theme auditor
  on every push (GitHub Actions).
- New nib logo; gallery layout; mtime cache-busting on admin assets.
